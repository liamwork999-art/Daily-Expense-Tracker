# 💸 Daily Expense Tracker

A simple JavaScript project that helps you track your daily spending, calculate totals, and understand where your money goes.

## ✨ Features

- Add daily expenses
- Calculate total spending
- Find the highest expense
- Count expenses above a chosen limit
- View average spending

## 🎯 What This Project Is For

This project is a practical practice exercise that simulates a real-life budgeting tool. It helps improve JavaScript skills such as:

- arrays
- loops
- conditionals
- functions
- math calculations

## ▶️ How to Run

1. Open the project folder in your editor.
2. Run the JavaScript file with Node.js:

```bash
node daily_expense_tracker.js
```

## 📊 Example Usage

You can test the program by entering a list of expenses such as:

```js
[12, 35, 8, 50, 20, 15];
```

The program can then show:

- total spending
- largest expense
- number of expenses above a limit
- average expense
